## 🛠️ Setup & Run Locally

1. **Clone the repo:**

   ```bash
   git clone https://github.com/sakthi2434/Portfolio.git
   cd Portfolio
````

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the project:**

   ```bash
   ng serve
   ```

4. **Visit:**

   Open your browser and go to `http://localhost:4200/`
